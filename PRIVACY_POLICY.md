# Политика конфиденциальности FinPlans

Дата последнего обновления: 11 июля 2026 г.

## 1. Введение
Добро пожаловать в FinPlans («Приложение»). Я, Виталий, являюсь разработчиком данного программного обеспечения. Защита ваших персональных данных является моим главным приоритетом.

## 2. Сбор и хранение данных
*   **Локальное хранение:** Приложение FinPlans работает по принципу «Local-First». Это означает, что абсолютно все ваши финансовые данные (счета, транзакции, цели накопления) хранятся исключительно в зашифрованном виде на вашем устройстве в локальной базе данных SQLite.
*   **Отсутствие облачной синхронизации:** На текущем этапе разработки мы не используем удаленные серверы для хранения или обработки вашей информации. Ваши данные никогда не покидают пределы вашего смартфона без вашего явного действия.

## 3. Передача данных третьим лицам
*   Мы **не собираем**, **не продаем** и **не передаем** ваши личные данные сторонним компаниям, рекламным сетям или аналитическим сервисам в фоновом режиме.
*   Единственная функция, которая может инициировать передачу данных во внешнюю сеть — это использование модуля «AI-консультант» (в будущих версиях). При использовании этой функции обезличенный фрагмент текста вашего бюджета отправляется внешнему нейросетевому API для генерации совета. Данные не сохраняются на стороне провайдера AI-сервиса дольше, чем требуется для ответа.

## 4. Резервное копирование
Функция экспорта данных создает JSON-файл с вашей финансовой историей. Этот файл сохраняется в памяти *вашего* устройства. Разработчик приложения не имеет доступа к этому файлу. Ответственность за безопасное хранение экспортированного файла лежит на пользователе.

## 5. Безопасность
Для защиты доступа к приложению планируется внедрение биометрической аутентификации (отпечаток пальца / Face ID), которую пользователь сможет включить по собственному желанию в настройках.
