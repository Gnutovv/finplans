FinPlans — Умный финансовый компаньон

![License](https://img.shields.io/badge/License-MIT-yellow.svg)
![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)

    FinPlans — это кроссплатформенное мобильное приложение для учета личных финансов, разработанное на чистом Dart/Flutter. Главная цель проекта — помочь пользователю перейти от простого наблюдения за тратами к осознанному планированию бюджета и достижению финансовых целей.

🎯 Основные возможности

    Управление капиталом: Создание счетов разных типов (простые, кредитные, вклады) с гибкими настройками процентных ставок.
    Гибкая аналитика транзакций: Каждая операция имеет необязательный параметр важности (Обязательные, Важные, Удовольствия). Это позволяет строить точные прогнозы накоплений, отсекая из расчетов случайные траты.
    Система тегов вместо жестких категорий: Пользователь сам управляет классификацией расходов. Одна транзакция может иметь несколько меток для многомерного анализа.
    «ФинПланы» — постановка целей: Главный модуль приложения. Укажите цель («Автомобиль») и сумму, а алгоритм рассчитает комфортный ежемесячный взнос, учитывая ваши обязательные расходы и доходы.
    Локальная безопасность: Приложение работает полностью автономно без доступа в интернет. Все данные хранятся в зашифрованной SQLite-базе (Drift).
    Свобода перемещения: Полноценный экспорт и импорт всех данных (счета, теги, история операций) в формате JSON для резервного копирования или переноса между устройствами.

🏗 Архитектура и технологии

Проект построен по принципам чистой архитектуры (Clean Architecture) с разделением на слои, что обеспечивает масштабируемость и легкость тестирования:

    State Management: BLoC + Freezed (строго типизированные состояния событий и UI).
    Локальная база данных: Drift (SQLite) со сложными SQL-запросами для построения отчетов средствами БД.
    Навигация: go_router (поддержка вложенных роутов и модальных окон).
    Инверсия зависимостей: get_it.
    Качество: Статический анализ very_good_analysis и покрытие логики Unit-тестами.

🤖 Будущее развитие (Roadmap)

    Интеграция AI-консультанта (GigaChat API): автоматические советы по оптимизации бюджета на основе агрегированных данных о тратах.
    Семейный доступ и синхронизация данных через бэкенд.
    Автоматическая категоризация транзакций на базе ML на устройстве.

🛠 Как запустить проект

    Клонируйте репозиторий: git clone https://github.com/Gnutovv/finplans.git
    Установите зависимости: flutter pub get
    Запустите на симуляторе или устройстве: flutter run
